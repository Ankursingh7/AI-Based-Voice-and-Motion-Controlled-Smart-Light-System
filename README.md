# 💡 AI-Based Voice Controlled Smart Light System

> Smart IoT lighting system powered by **AI + ESP32 + Voice Control**

---

## 🚀 Overview

An intelligent IoT-based home automation system that allows users to control lights using:

* 🎤 Voice Commands (Google Assistant / Alexa)
* 📱 Mobile Applications
* 🌐 Web Dashboard

Built using **ESP32**, the system enables remote control, automation, and energy-efficient lighting.

---

## ✨ Features

* 🔊 Voice-controlled light switching
* 🌐 Remote access via WiFi
* ⏰ Smart scheduling (time-based automation)
* 🧠 Habit learning (AI-based patterns)
* ⚡ Energy optimization
* 🔐 Secure authentication
* 📈 Scalable architecture

---

## 🏗️ System Architecture

```
User (Voice/App)
        ↓
   Internet / WiFi
        ↓
      ESP32
        ↓
     Relay Module
        ↓
       Light
```

---

## ⚙️ How It Works

1. User gives command (voice/app/web)
2. Command is sent via WiFi
3. ESP32 processes the request
4. Relay switches the light ON/OFF
5. Status is updated in real-time
